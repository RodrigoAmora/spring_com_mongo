# spring_com_mongo
Descrição:
----------
Projeto Web de Java com MongoDB, usando o SpringBoot.

Dependências:
-------------
O projeto usa o Java 17 e as seguintes dependências:
* Srping Boot
* Google Maps
* DevTools
* Thymeleaf

Rodando o Projeto:
------------------
Para rodar cada projeto localmente, execute o comando no diretório raiz projeto:
```shell script
mvn spring-boot:run
